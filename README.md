### Client Lead App 
A simple and responsive web application for collecting client information through a clean, user-friendly form. The application is designed to make it easy for users to submit their details while ensuring that required information and terms are properly validated before submission.

## Features 
- Clean and modern user interface 
- Respnsive design for desktop and mobile devices 
- Client information collection form 
- Required field validation 
- Terms & Conditions acceptance before submission
- Easy to customize and extend

## Technology Used 
- HTML5
- CSS3
-JavaScript (Vanilla JS)
